# AIBot

AI-powered toy system built using ESP32 and iOS.

## Features
- Voice interaction
- Emotion detection
- Mobile control via iOS app
- Hardware integration using ESP32

## Tech Stack
- Swift (iOS)
- ESP32
- Bluetooth / WiFi communication
- Xcode

## Project Structure
AIBot
 ├── Fluffbotics (iOS App)
 ├── firmware (ESP32 code)
 ├── hardware (circuit diagrams)
